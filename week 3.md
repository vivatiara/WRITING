- ### MENGAPA Array ada
  <div align="justify">Karena Array adalah tipe data list order yang dapat menyimpan tipe data apapun di dalamnya. Array dapat menyimpan tipe data String,      Number, Boolean, dan lainnya.

- ### Contoh Array
